COMMENT REMPLACER LES IMAGES

1. Glisse tes photos dans le bon sous-dossier images/
   - images/hero/       → ta photo de profil
   - images/sezame/     → shootings Sezame
   - images/mawu/       → visuels Mawu Paris
   - images/ecorce-wood/ → visuels Écorce & Wood
   - images/projets/    → logos, plaquettes, cartes de visite

2. Dans index.html, cherche les balises <img src="">
   et remplace le src par le chemin relatif, ex :
   src="images/hero/ma-photo.jpg"

3. Ouvre index.html dans ton navigateur pour vérifier.
